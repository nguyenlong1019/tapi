## Installaton  
1. Clone the project 
```bash
git clone https://github.com/nguyenlong1019/tapi.git 
```
2. Install dependencies:
```bash
pip install -r requirements.txt 
```
3. Setup environment variables
```bash
cp sample.env .env
```
4. Run the app 
```bash
python manage.py runserver
```
